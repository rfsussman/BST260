  1. Load the dslabs package and figure out what is in the us_contagious_diseases dataset. Create a data frame, call it avg, that has a column for year, and a rate column containing the cases of Measles per 10,000 people per year in the US. If a week was not report, it should not be included in the calculation of the rate.
# import packages
library(tidyverse)
library(dslabs)

# create avg dataframe
avg = us_contagious_diseases %>%
  filter(weeks_reporting > 0) %>% # filter out timepoints where there were no weeks reporting
  filter(disease == "Measles" & !state %in% c("Hawaii", "Alaska")) %>% # filter for measles, the 48 states
  group_by(year) %>% # want units per year
  summarise(rate = sum(count*(52/weeks_reporting))/sum(population)*10^4, .groups = "drop") # make rate variable, accounting for weeks reporting

According to the WHO, measles is a contagious and severe condition, so reducing the prevalence of measles in the general population is of critical public health importance. As shown in the graph above, from 1928 to 1962, measles cases were high: on average there were 35.701 cases per 10,000 individuals. But once the measles vaccine was deployed in 1963, measles cases dropped considerably: 2.413 cases per 10,000 individuals. Therefore, the measles vaccine is essential to preventing the disease and should be taken by all who can.

##MMWR: mortality and morbidity weekly report
library(lubridate)
mmwr_to_date = function(year, week){
  jan_4 = make_date(year = year, month = 1, day = 4)
  round_date(jan_4, unit = "week") + weeks(week-1) # sunday before Jan 4 is the first day of the MMWR year (wday(jan_4, label = TRUE))
}
